## Overview

This homelab is an experimentation platform for infrastructure patterns, container orchestration, local AI workloads, and network services.

## Hardware and Services

- Minisforum UM790 Pro and Raspberry Pi 5.
- Docker and k3s for container orchestration.
- Ollama for local LLM execution.
- PostgreSQL for data services.
- Pi-hole and Unbound for DNS and network control.

## Focus

The environment is used to work through practical infrastructure concerns: service deployment, container networking, local model hosting, persistent data, and operational patterns that are easier to learn by running real systems.
